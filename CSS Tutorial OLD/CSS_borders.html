The border-style Property

This property specifies what kind of border to display:

A dotted border.

A dashed border.

A solid border.

A double border.

A groove border.

A ridge border.

An inset border.

An outset border.

No border.

A hidden border.

A mixed border.

The border-radius Property

This property is used to add rounded borders to an element:

Normal border

Round border

Rounder border
